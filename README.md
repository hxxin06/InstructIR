# InstructIR
InstructIR 自然語言指令之多任務影像復原模型

影像還原旨在對受損影像進行深入分析與重建。傳統影像復原方法多專注於單一退化任務，例如獨立的去
噪、去霧或去模糊模型，但使用單一功能的模型難以應對複雜的真實環境，且缺乏與使用者互動的靈活性。
InstructIR模型是一個結合Transformer與自然語言處理的統一架構，模型透過文字指令（Text
Prompt）來識別使用者修復需求，並根據指令內容動態調整模型行為，不再侷限於單一功能，而是能在同一
個模型結構下完成多項任務，大幅提升了處理效能與泛化能力。

1.clone InstructIR 項目並安裝

2.配置模型

3.資料前處理

4.文字特徵提取

5.模型推論

6.進行修復任務並顯示結果
